This stack demonstrates all of the different types of templates supported in the Stacks API.

Each Scope Templates

Basic templates with each scope. HTML and CSS Tempaltes are also used in edit mode. All of these templates have access to the values from custom controls set in the sidebar for each stack instance.
  • html - HTML content. (edit mode)
  • css - css content. (edit mode)
  • js - Javascript content.

Page Scope Templates

Basic templates with page scope. These templates are placed only once on the page no matter how many stack instances appear on that page. These templates do NOT have access to values in custom controls.
  • css - css content. (edit mode)
  • js - Javascript content.

Page Scope HTML Templates

When adding page scoped HTML you can choose where the HTML will be added to the page.
  • type=html scope=pagePrefix - page prefix (edit mode)
  • type=html scope=pageHeadAbove - above head (edit mode)
  • type=html scope=pageHeadBelow - below head (edit mode)
  • type=html scope=pageBodyAbove - above body (edit mode)
  • type=html scope=pageBodyBelow - below body (edit mode)
